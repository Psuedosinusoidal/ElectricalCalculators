Variable Naming Convention:
  Standard:
    var_  : Variable, Numbers
    str_  : String, Letters
    fun_  : Function

  GUI:
    gui_  : Anything (and I mean ANYTHING) in the window (goes first)
    btn_  : A button, duh
    sep_  : Separator
    frm_  : Frame
    lbl_  : Label
    ent_  : Entry
    txt_  : Textbox
    stat_ : Status Text
    spin_ : Spinbox
    chk_  : Checkbutton/Checkbox

Version Naming Convention:
  x.y.z

  x = Major: A huge leap in the software.
  y = Minor: New features and/or bugfixes
  z = Build: A small change during development.
